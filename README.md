# Файлы для итогового задания

В директории `tests` находятся тесты для проверки API, которое должно быть реализовано в веб-сервере.

Директория `web` содержит файлы фронтенда.

Данный проект представляет собой менеджер задач позволяющий добавлять, редактировать, удалять задачу.
При добавлении/редактировании задачи действуют правила вычисления следующей даты работы с задачей.
После выполнения задачу нужно отметить выполненной. Если она была создана без правил повторения,
то переносится на следующую дату. Если нет - просто удаляется.

Для запуска проекта нужно:
1. скомпилировать приложение командой go build -o app.exe main.go
2. запустить ./app.exe

Для работы проекта используются следующие переменные среды в env файле:
Путь к веб директории
WEB_DIR_PATH: web
Порт запуска
PORT: "7540"
Путь к бд
DB_FILE: "scheduler.db"
Максимальное количество дней для переноса задачи
MAX_INTERNAL_DAYS: 400

Для запуска тестов использовать команду go test ./tests
При запуске тестов используются параметры в файле tests/settings.go
Параметры FullNextDate, Search можно включить. Реализован весь функционал.
